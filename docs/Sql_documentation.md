# Query 1 – Total Revenue

## Business Question

How much total revenue has the company generated?

---

## SQL Query

```sql
SELECT
    SUM(sales_amount) AS total_revenue
FROM fact_sales;
```

---

## Concepts Used

- SELECT
- FROM
- SUM()
- AS
- Aggregate Function

---

## Explanation

This query calculates the total revenue generated by all sales.

`SUM()` adds every value in the `sales_amount` column and returns a single result.

`AS total_revenue` renames the output column to make it more readable.

Since no `GROUP BY` clause is used, PostgreSQL aggregates all rows into one total.

## Interview Questions

### What is an aggregate function?

An aggregate function performs a calculation across multiple rows and returns a single value.

Examples include:

- SUM()
- AVG()
- COUNT()
- MAX()
- MIN()

---

### Why is GROUP BY not required?

Because the query returns a single aggregated value for the entire table rather than separate values for different groups.

---

### Does SELECT modify the database?

No.

SELECT only retrieves data without changing the database.
